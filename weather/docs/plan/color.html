Background color

For this site I used two different background colors. The header and footer have a basic black background. The body however uses a dark grey. The hex code for the grey is #24252a

text color

I used three different text colors in this project. For the navigation bar, I used the preprogrammed "Whitesmoke" color.
For the headings, I used rgb(233, 233, 109) which is a yellow color.
Lastly, I used "Aliceblue" for the text on the page.

Buttons and Links

The button and links use the same color. The links use it as the hover color while the button has it in the background. The color I chose is #0088a9. This is a coral blue color. I thought it made a great accent color.
